## 功能特色

- **雙縣市支援**：台北市（照片黏貼成冊）與新北市（上傳消防局平台）分流處理
- **白板填寫提示**：拍照前即時顯示案場名稱、測試人員、測試項目、測試位置與日期，減少漏填
- **入鏡確認提醒**：逐張提示測試人、監造人、標示牌入鏡狀態
- **日期浮水印**：拍照後自動於照片右下角加入西元日期（如 `2026.03.29`），與標示牌日期一致
- **自動命名**：依審查順序表規則自動產生照片檔名，新北市並對應官方照片表欄位
- **設備資料夾分類**：各設備（室內消防栓、自動撒水、排煙設備…）獨立資料夾，ZIP 打包下載
- **歷史案件記憶**：最多保存 5 筆歷史案件資料，下拉選單一鍵帶入，減少重複填寫
- **多人密碼管理**：各人員使用獨立密碼登入，畫面顯示使用者姓名
- **純前端・零後端**：單一 HTML 檔案，照片僅存於瀏覽器記憶體，不上傳任何伺服器

## 支援設備項目

**台北市**：共有設備 / 水系統滅火設備 / 警報設備 / 避難逃生設備 / 消防搶救必要設備

**新北市**：依「建築物消防安全設備安裝施工測試照片審查順序表」全項目，包含：
- 發電機・耐燃耐熱保護
- 滅火器 / 室內(外)消防栓 / 自動撒水 / 水道連結型自動撒水 / 泡沫 / 水霧
- 火警自動警報 / 火警發信警報 / 緊急廣播
- 出口標示燈 / 避難方向指示燈 / 緊急照明 / 避難器具 / 緩降機 / 救助袋
- 連結送水管 / 專用蓄水池 / 室內排煙（機械・自然）/ 梯間排煙 / 緊急電源插座 / 無線電通信輔助
- 化學系統滅火（乾粉・CO₂・惰性氣體・鹵化烴・簡易自動滅火）
- 其他補充相片

## 使用方式

1. 開啟網址，輸入個人密碼登入
2. 選擇縣市（台北市 / 新北市）
3. 填寫案件資訊（支援帶入歷史案件）
4. 勾選本次需檢附的設備項目
5. 依序拍照（相機直拍或選取相簿檔案）
6. 下載 ZIP（含各設備資料夾）或匯出檔名清單

## 密碼管理

在 `index.html` 頂部找到 `const USERS = {...}` 區塊，直接新增或修改：

```javascript
const USERS = {
  "wu-2026":   "吳○○ 消防設備師",
  "qin-2026":  "秦○○ 消防設備士",
};
```

每次異動後重新 commit 即生效。

---

## 注意事項

- 照片僅存於瀏覽器記憶體，**關閉分頁後消失**，請務必在工作階段內完成下載
- 密碼以明文儲存於原始碼，適合內部小團隊使用；如需更高安全性請搭配後端驗證
- 新北市照片表以 110 年 10 月 18 日修訂版為基準，如有再修正請對應更新項目清單
